Ok, so now that we've got it installed and we have the credentials sorted, let's instantiate it so we can actually use it!

var dynasty = require('dynasty')(credentials);

Optionally, provide a url as the second argument to connect to a Dynamo instance that is not on AWS. For example if you are running Dynamo locally you would instantiate it as follows:

var dynasty = require('dynasty')(credentials, 'localhost:8000');

Note, you still have to provide credentials, even if running locally doesn't require them because the internal code expects them to be present.
